## 🔍 Key Projects

### 1️⃣ **Car Price Prediction**
- **Description**: Predicts the price of a car based on its features (e.g., mileage, year, brand, fuel type).
- **Tech Stack**: 
  - Regression Models (Linear, Ridge, Lasso)
  - Data Preprocessing and Feature Engineering
- **Output**: Price prediction based on user inputs.
- **Status**: ✅ Completed & Deployed

---

### 2️⃣ **Crop Prediction**
- **Description**: Recommends the best crop to grow based on environmental conditions such as soil nutrients and weather.
- **Tech Stack**:
  - Random Forest Classifier
  - Gradient Boosting and Feature Scaling
- **Visuals**: Heatmaps and distribution plots for better insights.
- **Output**: Optimal crop suggestions tailored to conditions.
- **Status**: ✅ Completed & Deployed

---

### 3️⃣ **Personalized Medicine**
- **Description**: Suggests the correct medicine dosage based on patient data (e.g., genetics, age, weight).
- **Tech Stack**:
  - Advanced Regression Models
  - Feature Scaling (Standard and MinMax Scalers)
- **Output**: Dosage recommendations for personalized treatment.
- **Status**: ✅ Completed & Deployed

---

### 4️⃣ **Predicting Diabetes**
- **Description**: Predicts the risk of diabetes based on medical data such as glucose levels, BMI, and blood pressure.
- **Tech Stack**:
  - Logistic Regression, Random Forest Classifier
  - Hyperparameter Tuning for Improved Accuracy
- **Output**: Risk assessment for diabetes.
- **Status**: ✅ Completed & Deployed

## 🌐 Deployment

Each project is equipped with a **Flask-based web application** that offers a user-friendly interface for real-time predictions.

### 🛠️ How to Run Locally:
1. **Clone this Repository**:
   ```bash
   git clone https://github.com/adesh-pawar10/AI-ML-Projects.git
   cd AI-ML-Projects
   ```
2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the Application**:
   ```bash
   python app.py
   ```
4. **Access in Browser**:
   Open `http://127.0.0.1:5000/` to use the application.
